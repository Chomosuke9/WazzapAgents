# Tips & Praktik Terbaik

## Untuk Admin Grup

1. **Selalu coba di grup testing dulu** sebelum mengaktifkan moderasi di grup sungguhan
2. **Mulai dari permission level rendah** (0 atau 1), baru naikkan kalau sudah yakin
3. **Buat prompt yang jelas tentang batasan** — bot yang tidak tahu batasnya bisa bertindak berlebihan
4. **Gunakan `/reset` setelah mengubah prompt besar-besaran** agar bot tidak bingung dengan konteks lama
5. **Monitor grup secara aktif** terutama di awal penggunaan moderasi

## Untuk Menulis Prompt Moderasi

- Sebutkan dengan **eksplisit** kondisi kapan harus kick vs hanya mengingatkan
- Berikan **sistem peringatan** sebelum kick (contoh: 3-5 peringatan dulu)
- Definisikan dengan jelas apa itu "toksik" atau "spam" menurut standar grupmu
- Hindari instruksi ambigu seperti "hapus semua pesan jelek" — bot bisa salah interpretasi

## Untuk Pengguna Biasa

- Bot paling baik merespons jika kamu **mention atau reply** langsung ke bot
- Kirim **satu topik per pesan** untuk hasil terbaik
- Jika bot memberikan jawaban tidak relevan, coba **ketik ulang** pertanyaanmu dengan lebih spesifik
- Gunakan `/reset` jika percakapan sudah sangat kacau
